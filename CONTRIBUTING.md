## Contributing to Clash M

#### Code Style

Please use `Android Studio` or `Intellij IDEA` to open the project and use the project code style profile.

`File` -> `Settings` -> `Editor` -> `Code Style` -> `C/C++ and Kotlin` -> `Scheme` -> `Project`
